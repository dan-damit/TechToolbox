## about_Remove-Printers

### SHORT DESCRIPTION
Removes all printers from the system, with optional removal of ports, drivers, and per-user mappings.

### LONG DESCRIPTION
The Remove-Printers function uses Win32_Printer (CIM) to remove printer queues after resetting the print spooler service and clearing the spool folder. It provides options to also remove TCP/IP ports and printer drivers. The function includes fallbacks for provider hiccups and frees common process locks such as splwow64 and PrintIsolationHost. Additionally, it can remove per-user network printer connections across all user profiles.

### PARAMETERS
- **IncludePorts**: Also removes TCP/IP printer ports (non-standard).
- **IncludeDrivers**: Also removes printer drivers after the queues are gone.
- **Force**: Performs a best-effort forced cleanup of driver packages via pnputil if standard removal fails.
- **AllUsers**: Attempts to remove per-user network printer connections for all user profiles.
- **PassThru**: Outputs a summary object with counts and failures.

### EXAMPLES
Remove-Printers -IncludePorts -IncludeDrivers -Force -AllUsers -PassThru

Remove-Printers -WhatIf
