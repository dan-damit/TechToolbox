### LONG DESCRIPTION
Explains the function's behavior: generates a password via `Get-NewPassword`, applies it via `Set-ADAccountPassword`, and optionally unlocks the account, forces change-at-logon, or delivers plaintext via console/file/clipboard. - `-Style` - Password generation style: 'Random', 'Readable', or 'Passphrase' (default: 'Readable')
- `-OutFile` - Path to write plaintext password to file
- `-Clipboard` - Copies plaintext password to Windows clipboard
- `-Credential` - Reserved for module interface; passed through to AD cmdlets
