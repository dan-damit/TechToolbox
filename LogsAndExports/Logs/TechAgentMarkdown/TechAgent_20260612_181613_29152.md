
### Recursive Search Across a Tree
To find these files anywhere inside a larger directory structure (like your entire TechToolbox repository):
```powershell
Get-ChildItem -Recurse -Path "C:\repos\TechToolbox" -Filter "about_*.help.txt" -File
```

### Key Flags Explained
- `-Recurse`: Searches all subdirectories within the provided path.
- `-Filter`: Uses native file system filtering (faster than `-Include` for exact wildcard patterns).
- `-File`: Ensures only files are returned, ignoring any directories that might accidentally match.
- `| Format-Table Name, FullName, Length`: (Optional pipe) Formats the output into a neat table showing just the file name, full path, and size in bytes.
