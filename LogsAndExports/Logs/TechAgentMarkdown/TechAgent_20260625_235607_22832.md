.EXTERNALHELP Remove-Printers-help.xml

.NOTES
  NAME: about_Remove-Printers

.DESCRIPTION
  The Remove-Printers function removes all printers from the system. It resets
  the print spooler service and clears the spool folder to ensure a clean state.
  By default, it only removes printer queues for the current user or local machine,
  but can be configured to remove TCP/IP ports, printer drivers, and per-user
  network connections across all profiles on the system.

.PARAMETER IncludePorts
  When specified, also removes TCP/IP printer ports associated with the removed
  printers. This is useful for a complete cleanup of printer-related components.

.PARAMETER IncludeDrivers
  When specified, also removes printer drivers after the printer queues have been
  deleted. This helps in freeing up disk space and removing unused driver packages.

.PARAMETER Force
  Enables best-effort forced cleanup of driver packages using pnputil if standard
  removal methods fail. This parameter is useful when dealing with stubborn or
  corrupted driver installations.

.PARAMETER AllUsers
  When specified, attempts to remove per-user network printer connections for all
  user profiles on the system. This ensures that no residual printer mappings
  remain for any user account.

.PARAMETER PassThru
  When specified, outputs a summary object containing counts of removed printers,
  ports, drivers, and any failures encountered during the process. This is useful
  for auditing or logging purposes.

.INPUTS
  None. You cannot pipe objects to Remove-Printers.

.OUTPUTS
  [pscustomobject]. If -PassThru is specified, returns a summary object with
  removal statistics and failure details.

.EXAMPLE
  Remove-Printers -IncludePorts -IncludeDrivers -Force -AllUsers -PassThru

  Description:
  Removes all printers, TCP/IP ports, drivers, and per-user connections for all
  users on the system. Forces driver cleanup if standard removal fails and outputs
  a summary object.

.EXAMPLE
  Remove-Printers -WhatIf

  Description:
  Shows what would happen if the command were executed without actually removing
  any printers or components.
