.EXTERNALHELP Remove-Printers-help.xml

.NOTES
    Function: Remove-Printers
    Source: C:\repos\TechToolbox\Public\System\Remove-Printers.ps1
    Description: This function is part of the TechToolbox suite.

.DESCRIPTION
    The Remove-Printers cmdlet removes all printers from the local system. It operates by resetting the print spooler service and clearing the spool folder to ensure a clean state before removing printer queues via Win32_Printer (CIM). 

    This function supports advanced cleanup options, including the removal of TCP/IP ports and printer drivers. It includes fallback mechanisms for common provider issues and frees process locks associated with printing (such as splwow64.exe and PrintIsolationHost.exe). Additionally, it can remove per-user network printer connections across all user profiles on the system.

.PARAMETER IncludePorts
    Specifies that TCP/IP printer ports should also be removed from the system. This is a non-standard operation and should be used with caution as it may affect other services relying on these ports.

.PARAMETER IncludeDrivers
    Specifies that printer drivers should be removed after all printer queues have been deleted. This helps in cleaning up unused driver packages from the system.

.PARAMETER Force
    Enables best-effort forced cleanup of driver packages using pnputil if standard removal methods fail. This parameter is useful for stubborn drivers that do not remove cleanly through normal means.

.PARAMETER AllUsers
    Specifies that per-user network printer connections should be removed for all user profiles on the system, not just the current user. This ensures a comprehensive cleanup across all accounts.

.PARAMETER PassThru
    Specifies that a summary object containing counts of printers/ports/drivers removed and any errors encountered should be output to the pipeline. Without this parameter, no output is generated upon success.

.INPUTS
    None. You cannot pipe objects to Remove-Printers.

.OUTPUTS
    System.Management.Automation.PSCustomObject (if -PassThru is specified).
    A summary object containing:
    - PrintersRemoved: Count of printers removed.
    - PortsRemoved: Count of ports removed (if -IncludePorts used).
    - DriversRemoved: Count of drivers removed (if -IncludeDrivers used).
    - Errors: List of any errors encountered during the process.

.EXAMPLE
    Remove-Printers -IncludePorts -IncludeDrivers -Force -AllUsers -PassThru

    This command removes all printers, TCP/IP ports, and printer drivers from the system. It forces cleanup of stubborn drivers and removes per-user connections for all users. A summary object is returned detailing what was removed.

.EXAMPLE
    Remove-Printers -WhatIf

    This command simulates the removal of all printers without actually performing any changes. It displays a preview of what would happen if the cmdlet were executed.
