## about_Remove-Printers

### SHORT DESCRIPTION
Removes all printers from the system.

### LONG DESCRIPTION
The Remove-Printers function removes all printer queues from the local system. It uses Win32_Printer (CIM) to remove queues after resetting the spooler and clearing the spool folder.

Optionally, it can remove TCP/IP ports and printer drivers. The function adds fallbacks for provider hiccups and frees common process locks (splwow64/PrintIsolationHost). It can also remove per-user network printer connections across all user profiles if specified.

### SYNTAX
Remove-Printers [-IncludePorts] [-IncludeDrivers] [-Force] [-AllUsers] [-PassThru] [<CommonParameters>]

### PARAMETERS
#### -IncludePorts
Also remove TCP/IP printer ports (non-standard).

    Position?                    named
    Default value                False
    Accept pipeline input?       False
    Accept wildcard characters?  false

#### -IncludeDrivers

#### -Force

    Accept wildcard characters?  false

#### -AllUsers

#### -PassThru

### EXAMPLES
#### Example 1

This command removes all printers, ports, and drivers from the system for all users, forces cleanup of driver packages, and outputs a summary.
