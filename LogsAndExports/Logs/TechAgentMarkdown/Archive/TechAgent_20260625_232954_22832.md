## about_Remove-Printers

.NAME
    about_Remove-Printers

.SYNOPSIS
    Removes all printers from the system, with optional removal of ports,
    drivers, and per-user mappings.

.DESCRIPTION
    The Remove-Printers function uses Win32_Printer (CIM) to remove printer queues after resetting the print spooler service and clearing the spool folder. It optionally removes TCP/IP ports and printer drivers. The function includes fallbacks for provider issues and frees common process locks such as splwow64 and PrintIsolationHost. It can also remove per-user network printer connections across all user profiles.

.PARAMETER IncludePorts
    Also remove TCP/IP printer ports (non-standard).

.PARAMETER IncludeDrivers
    Also remove printer drivers after the queues are gone.

.PARAMETER Force
    Best-effort forced cleanup of driver packages via pnputil if standard removal fails.

.PARAMETER AllUsers
    Attempt to remove per-user network printer connections for all user profiles.

.PARAMETER PassThru
    Output a summary object with counts and failures.

.INPUTS
    None. You cannot pipe objects directly to Remove-Printers.

.OUTPUTS
    System.Management.Automation.PSCustomObject (if -PassThru is used)

.EXAMPLE
    Remove-Printers -IncludePorts -IncludeDrivers -Force -AllUsers -PassThru

    Description:
    Removes all printers, TCP/IP ports, and drivers from the local system,
    clears per-user connections for all users, forces driver cleanup if needed,
    and outputs a summary object.

.EXAMPLE
    Remove-Printers -WhatIf

    Description:
    Shows what would happen if the command were executed without making any changes.
