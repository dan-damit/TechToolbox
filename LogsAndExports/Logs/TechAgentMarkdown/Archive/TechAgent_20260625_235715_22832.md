## about_Remove-Printers

TOPIC
    about_Remove-Printers

SHORT DESCRIPTION
    Removes all printers from the system, with optional removal of ports,
    drivers, and per-user mappings.

LONG DESCRIPTION
    Remove-Printers uses Win32_Printer (CIM) to remove printer queues after
    resetting the print spooler service and clearing the spool folder. It can
    optionally remove TCP/IP ports and printer drivers. The function includes
    fallbacks for provider issues and frees common process locks such as
    splwow64 and PrintIsolationHost. It can also remove per-user network
    printer connections across all user profiles on the system.

SYNTAX
    Remove-Printers [-IncludePorts] [-IncludeDrivers] [-Force] [-AllUsers]
        [-PassThru] [<CommonParameters>]

PARAMETERS
    -IncludePorts
        Also remove TCP/IP printer ports (non-standard).

        Type: SwitchParameter
        Required: False
        Default value: None
        Accept pipeline input: False

    -IncludeDrivers
        Also remove printer drivers (after queues are gone).

        Type: SwitchParameter
        Required: False
        Default value: None
        Accept pipeline input: False

    -Force
        Best-effort forced cleanup of driver packages via pnputil if standard
        removal fails.

        Type: SwitchParameter
        Required: False
        Default value: None
        Accept pipeline input: False

    -AllUsers
        Attempt to remove per-user network printer connections for all user
        profiles.

        Type: SwitchParameter
        Required: False
        Default value: None
        Accept pipeline input: False

    -PassThru
        Output a summary object with counts and failures.

        Type: SwitchParameter
        Required: False
        Default value: None
        Accept pipeline input: False

EXAMPLES
    Example 1:
        Remove-Printers -IncludePorts -IncludeDrivers -Force -AllUsers -PassThru

        Description
        -----------
        Removes all printers, TCP/IP ports, and drivers with forced cleanup,
        removes per-user connections for all users, and outputs a summary object.

    Example 2:
        Remove-Printers -WhatIf

        Description
        -----------
        Simulates the removal of all printers without making any changes.

NOTES
    - The function has high ConfirmImpact and supports ShouldProcess (WhatIf/Confirm).
    - It relies on internal helper functions for spooler management and process
      lock handling.
