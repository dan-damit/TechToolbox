## Clear-BrowserProfileData

### Synopsis
Clears Chromium browser profile data for Chrome and/or Edge.

### Description
Performs a browser profile cleanup workflow for Google Chrome, Microsoft Edge, or both. The function loads runtime configuration from `settings.browserCleanup` in config.json to apply defaults for parameters not explicitly supplied by the caller. It resolves target browsers (Chrome, Edge, or All), optionally stops browser processes to avoid file locks, discovers Chromium profile folders, and clears cookies, cache, and local storage as configured.

This function supports `-WhatIf` and `-Confirm`. Process termination and deletion work are guarded by ShouldProcess-aware flow.

### Parameters

- **Browser**
  - Type: `string`
  - Default: `'All'`
  - Valid values: `'Chrome'`, `'Edge'`, `'All'`
  - Description: Specifies which browser(s) to clear. Defaults to 'All'. Can be overridden by config.

- **Profiles**
  - Type: `string[]`
  - Default: `$null` (all profiles)
  - Description: Array of specific profile names to target (e.g., `'Default'`, `'Profile 1'`). If omitted, all discovered profiles are processed. Can be overridden by config.

- **IncludeCookies**
  - Type: `bool`
  - Default: `$true`
  - Description: Whether to clear cookies. Defaults to true. Can be overridden by config.

- **IncludeCache**
  - Type: `bool`
  - Default: `$true`
  - Description: Whether to clear cache. Defaults to true. Can be overridden by config.

- **SkipLocalStorage**
  - Type: `bool`
  - Default: `$false`
  - Description: If `$true`, local storage data is preserved. Defaults to false (local storage cleared). Can be overridden by config.

- **KillProcesses**
  - Type: `bool`
  - Default: `$true`
  - Description: Whether to stop browser processes before deletion to avoid file locks. Defaults to true. Can be overridden by config.

- **SleepAfterKillMs**
  - Type: `int`
  - Default: `1500`
  - Description: Milliseconds to wait after killing processes before proceeding with cleanup. Defaults to 1500ms. Can be overridden by config.

### Inputs
None. The function does not accept pipeline input.

### Outputs
None. The function performs actions but does not return objects.

### Examples

#### Example 1: Clear all data for all browsers
```powershell
Clear-BrowserProfileData
```
This command clears cookies, cache, and local storage for all Chrome and Edge profiles, killing processes if necessary.

#### Example 2: Clear only Chrome cache
```powershell
Clear-BrowserProfileData -Browser Chrome -IncludeCookies $false -SkipLocalStorage $true
```
This command targets only Google Chrome, clears only the cache (skipping cookies and local storage), and does not kill processes if they are running.

#### Example 3: Clear specific profile for Edge with confirmation
```powershell
Clear-BrowserProfileData -Browser Edge -Profiles 'Default' -Confirm
```
This command targets the 'Default' profile in Microsoft Edge and prompts for confirmation before proceeding.
